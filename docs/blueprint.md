# **App Name**: Shariah Standards Pro

## Core Features:

- Course Information Display: Display comprehensive details about the CPSS exam preparation course, including scope, fees, and schedule.
- Trainer Profiles: Showcase expert trainers with their qualifications and experience in Islamic finance.
- Registration and Enrollment: Enable users to register for the course and download the course outline.
- Curriculum Suggestion Tool: AI tool that takes as input user goals and current experience to then suggest the right courses for that person.

## Style Guidelines:

- Primary color: Deep navy blue (#0F2A44) to convey professionalism and trust, aligning with the financial sector.
- Background color: Very light desaturated blue (#EBF0F5) for a clean, modern backdrop that complements the primary color.
- Accent color: Gold/yellow (#F4C430) to highlight key elements and calls to action, providing a touch of elegance.
- Body and headline font: 'PT Sans', sans-serif, for a modern yet readable aesthetic. The font will be used in both headlines and body.
- Use consistent, professional icons that reflect Islamic finance principles.
- Implement a grid-based layout with rounded cards and soft shadows for a premium look.
- Incorporate subtle fade-in and slide-up animations for a smooth user experience.
